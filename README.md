# PCOD Tracker

## Overview
The PCOD Tracker is a web application designed to help users monitor and manage symptoms associated with Polycystic Ovary Syndrome (PCOS). The application allows users to log their symptoms, track changes, and gain insights into their health over time.

## Features
- **User-friendly Interface**: Built with HTML and JavaScript to provide an easy-to-use experience.
- **Symptom Logging**: Users can enter and track various symptoms related to PCOS.
- **Health Insights**: Provides analytics and charts to help users understand their symptoms over time.
- **Responsive Design**: Works on both desktop and mobile devices.

## Technologies Used
- **HTML**: Structure of the web application.
- **JavaScript**: Add interactivity and dynamic content to the application.
- **CSS**: Styling the application for an appealing look.

## Getting Started
1. Clone the repository: `git clone https://github.com/Sakshisingh1105/pcod-tracker.git`
2. Open `index.html` in your browser to view the application.
